Cat shell
============
This is a Shell made using C and WinAPI that provides the main cmd/windows commands you'd expect from a shell, with special commands with special names, etc.

![shell Preview](https://s6.uupload.ir/files/screenshot_2025-12-06_004306_lgj4.png)

---

## Features
- Most of the cmd commands
- Special commands
- Cat

---

## Special Commands
Exclusive Commands:
- hey -> you can just say hello to shell.    
- clean -> this is just cls.    
- whereismycat -> this is pwd but with more fun.
- catfood -> showing folder and files like dir.
- mycatversion -> version of your shell
- catmeows -> history of your prevous commands
- help
>You can see a full list of not exclusive commands (cmd commands) in:
https://learn.microsoft.com/en-us/windows-server/administration/windows-commands/windows-commands

---

## Install
After you clone this repo to your desktop, go to its root directory and run `gcc main.c` or `g++main.c` to execute.
This shell is for **Windows** not linux or macOS.
